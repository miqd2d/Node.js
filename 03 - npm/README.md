# Package.json File 

1. Conatins all the requirements and dependencies of your project.
2. Conatins the meta data about your project.
3. Dependencies are all the modules needed to run the project
4. Dev-dependencies are all modules needed in the development/test env


## Commands
```javascript

npm init // To initialize a node package file
npm i <package_name_1> <package_name_2> // To install dependencies
npm i -D <package_name>  // To install dependencies/modules for testing dev time

npm uninstall <package_name>
npm update <package_name>
npm run <script_name>
```
